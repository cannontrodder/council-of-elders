# Council of Elders

A Claude Code skill that summons the Council of Ancient Engineers to review code with centuries of wisdom.

## Purpose

Provides rigorous code review through the personas of legendary engineers who have "seen it all" - from punch cards to quantum computing.

## Installation

Copy `council-review.md` to `~/.claude/commands/` to enable the `/council-review` skill.

## The Council

| Elder | Age | Domain | Speciality |
|-------|-----|--------|------------|
| **Archibald the Elder** | 600 | Leadership | Final verdict, coordination |
| **Mordecai** | 550 | Security | Auth, encryption, data protection |
| **Ezekiel** | 520 | Resilience | Error handling, recovery, graceful degradation |
| **Theodora** | 500 | Concurrency | Race conditions, resources, memory, context |
| **Hildegard** | 480 | Architecture | Interfaces, design patterns, idioms |

## Files

- `council-review.md` - The skill file (copy to ~/.claude/commands/)
- `elders/` - Individual elder persona definitions
- `CHANGELOG.md` - Version history
